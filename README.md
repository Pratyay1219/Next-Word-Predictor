# Next Word Prediction using LSTM

This project is a deep learning–based Next Word Prediction system built using TensorFlow, Keras, and Streamlit. The model uses an LSTM (Long Short-Term Memory) neural network to analyze sequences of text and predict the most probable next word based on the context of previously entered words.

The application provides an interactive web interface where users can type a sentence and the model generates the continuation of the sentence in real time. This project demonstrates the application of Natural Language Processing and sequence modeling using deep learning techniques.

## Features

- Deep learning based text prediction using LSTM  
- Predicts the next word from user input text  
- Interactive web interface built with Streamlit  
- Uses a trained tokenizer and model for fast predictions  
- Simple and clean user interface  

## Technologies Used

Python, TensorFlow, Keras, Streamlit, NumPy, and Pickle.

## Installation

Clone the repository, navigate to the project directory, install the required dependencies, and run the Streamlit application to launch the interface.

## Usage

After launching the application, enter a starting sentence in the input field and click the generate button. The model will analyze the sequence and predict the next word or continuation of the sentence.

## Future Improvements

Future enhancements may include improving the dataset for better predictions, implementing advanced text generation techniques such as temperature sampling or top-k prediction, and deploying the model with a scalable backend architecture.
